# H&I Lista de Precio - POS System 🚀

Sistema de gestión automatizado para comercios minoristas (bodegas) en Barquisimeto, Venezuela.

### Características Principales:
- **Multitenant:** Aislamiento total de datos por bodega mediante Firebase Auth.
- **Cálculo Automático:** Conversión de precios USD a VES en tiempo real según tasa dinámica.
- **Gestión de Inventario:** Ordenamiento alfabético automático y control de stock.
- **Exportación:** Generación de listas de precios en formato Excel (.xlsx).
- **Tech Stack:** TypeScript, React, Firebase Firestore, Vite.
